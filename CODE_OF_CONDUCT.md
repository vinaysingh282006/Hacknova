# Code of Conduct

## Our Commitment

EcoPulse is committed to providing a welcoming, inclusive, and harassment-free environment for all contributors, regardless of experience level, background, or identity.

---

## Expected Behavior

Contributors are expected to:

- Be respectful and considerate in all interactions
- Use inclusive and professional language
- Accept constructive feedback gracefully
- Collaborate in good faith
- Help create a positive community atmosphere

---

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment, discrimination, or offensive remarks
- Personal attacks, trolling, or intimidation
- Publishing private information without consent
- Disruptive behavior that harms collaboration
- Any conduct that is unprofessional or hostile

---

## Scope

This Code of Conduct applies to:

- GitHub issues, pull requests, and discussions
- Project-related communication channels
- Any interaction representing the EcoPulse project

---

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

Actions may include:
- Issuing warnings
- Temporarily or permanently restricting participation
- Rejecting or reverting contributions

All enforcement decisions will be fair and focused on maintaining a safe community.

---

## Reporting Issues

If you experience or witness unacceptable behavior, report it to the project maintainers privately through GitHub.

All reports will be handled with discretion.

---

---

By participating in EcoPulse, you agree to abide by this Code of Conduct.
